A lightweight and customizable floating cart button for WooCommerce. Enhance your store's user experience with a stylish, always-visible cart button.

== Description ==

**Floating Cart Button for WooCommerce** is a simple yet powerful plugin that adds a floating cart button to your WooCommerce store. The button stays visible as users scroll, making it easy for them to access their cart from anywhere on your site.

Key Features:
- **Floating Cart Button**: A stylish, always-visible cart button that stays fixed on the screen.
- **Customizable Design**: Change the button's position, icon color, and background color.
- **Real-Time Cart Updates**: The cart count updates dynamically as products are added or removed.
- **Hide on Cart/Checkout Pages**: Hidden the button on the Cart and Checkout pages.
- **Lightweight and Fast**: Built with performance in mind, ensuring it doesnâ€™t slow down your site.

Perfect for improving user experience and increasing conversions on your WooCommerce store!

== Installation ==

1. Upload the `floating-cart-button-for-woocommerce` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Go to **Cart Button** to configure the plugin.

== Frequently Asked Questions ==

= Does this plugin work with any WordPress theme? =
Yes, the plugin is designed to work with any WordPress theme that supports WooCommerce.

= Can I customize the appearance of the floating cart button? =
Yes, you can customize the button's position, icon color, and background color from the plugin's settings page.

= Does the plugin support real-time cart updates? =
Yes, the cart count updates dynamically as products are added or removed via AJAX.

= Can I hide the floating cart button on specific pages? =
The button is already hidden on the Cart and Checkout pages from the plugin's settings.
